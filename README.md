1. Install node(v18 or higher)

   ```sh
   brew install node
   
   ```

2. Install next.js and yarn

   ```sh
   npm install -g next@latest
   npm install -g yarn
   ```

3. Install dependencies

   ```sh
   yarn install
   ```

4. Start the development server

   ```sh
   yarn dev
   ```
